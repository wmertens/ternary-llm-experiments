# Ternary LLM experiments

This repo contains a bunch of experiments done with the help of Claude, to someday make a ternary-valued LLM from scratch.

It started with converting SmoLLM to ternary at the 135M scale, using a Bonsai-style layout. The best attempt uses the Hestia approach, but it's still pretty bad.

After that I experimented with various training methods to make a ternary model from scratch.
